# Page Setup

## Home

Use the store home page for the landing screen. The hero, announcement cards, and community cards are driven by the schema values.

If you want extra text below the home sections, put it in the Tebex home page description field.

## Rules page

- Title: `Rules`
- Slug: `rules`
- Content: optional. You can leave it empty or add extra server-specific notes.

## Vote page

- Title: `Vote`
- Slug: `vote`
- Content: optional. The layout already shows the heading and vote button area.
- Add your real vote URLs in the schema options `vote-link-1` through `vote-link-4`.

## Store page

- Title: `Store`
- Slug: `store`
- Content: optional. The layout already shows rank and money summaries plus custom order guidance.

## Staff page

- Title: `Staff`
- Slug: `staff`
- Content: paste the HTML from `content/staff-page.html` and replace the placeholder names with your real team.

## Recommended category slugs

- `ranks`
- `cosmo-money`
- `custom`

If you use different slugs, update them in the schema.
