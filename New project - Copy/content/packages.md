# Tebex Package Copy

Paste these into your Tebex package descriptions so the custom category and package pages show polished perk lists automatically.

### Ranks category description

```html

```

### Cosmo Money category description

```html

```

```html
<p>Need something unique? Open a Discord ticket first and staff will guide you through a custom order.</p>
```
